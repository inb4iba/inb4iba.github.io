Variáveis + Template Literals

Descrição do exercício:
  • Seguindo os padrões de nomenclatura, crie uma variável do tipo Number e outra do tipo String.
  • Crie uma ou mais variáveis e use estas em uma String por meio de concatenação ou Template Literal;

let thisIsAString = "Isso é uma string";

let numberOneTwoThree = 123;


console.log(`Variável 1: ${thisIsAString} | Variável 2: ${numberOneTwoThree}`);


//Resultado do console: Variável 1: Isso é uma string | Variável 2: 123
