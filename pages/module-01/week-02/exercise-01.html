Discursivas

Descrição do exercício:
  1. Com suas palavras descreva a relação entre ECMAScript e JavaScript. Escreva da maneira que você considerar mais fácil a diferença entre as linguagens Javascript e ECMAScript.
  2. Explique brevemente o método “console.log()”. Explique a sintaxe do método console.log() e o que ele faz.
Respostas:
  1. ECMAScript nada mais é do que o Javascript padronizado pela ECMA (European Computer Manufacturers Association), essa padronização aconteceu quando a Netscape enviou para eles, com o intuito de que os navegadores fossem padronizados. A diferença entre as 2 linguagens além do nome, é que o ECMAScript seria a especificação, contendo diversas funcionalidades exclusivas e o Javascript seria a linguagem base dessa implementação.
  2. O método console.log() imprimi no console os valores passados. No caso da sintaxe, chamamos a função log do objeto console, nele podemos passar qualquer tipo de dado, inclusive podemos mandar diversos dados separados por vírgulas.
